## 🎯 使い方

### 1. プロジェクトを作成

```
1. プロジェクト一覧の右上「新規作成」ボタンをクリック
2. プロジェクト名を入力
3. 「プロジェクトを作成」ボタンをクリック
4. 自動的にプロジェクト詳細画面へ遷移
```

### 2. プロジェクトを編集

```
1. プロジェクト詳細画面の右上「編集」ボタンをクリック
2. プロジェクト名やアーカイブ設定を変更
3. 「更新」ボタンをクリック
4. プロジェクト詳細画面へ戻る
```

### 3. タスクの詳細を見る

```
1. プロジェクト詳細画面でタスクカードをクリック
2. タスク詳細画面が表示される
3. 右上の鉛筆アイコンで編集モードに切り替え
4. タイトル・説明・ステータスを編集
5. 「変更を保存」ボタンをクリック
```

### 4. タスクを削除

```
1. タスク詳細画面の右上ゴミ箱アイコンをクリック
2. 確認ダイアログで「OK」をクリック
3. プロジェクト詳細画面へ戻る
```
