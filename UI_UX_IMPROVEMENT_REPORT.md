## 🎨 デザインの特徴

### グラスモーフィズム
```css
backdrop-blur-lg bg-white/70
border border-white/50
```

### グラデーション
- ヘッダーアイコン: `from-blue-500 to-indigo-600`
- アクティブボタン: `from-blue-500 to-indigo-500`
- タイトル: `from-blue-600 to-indigo-600`

### アニメーション
- ホバー時の影の拡大
- スムーズなトランジション（300ms）
- 作業中タスクの点滅エフェクト

### レスポンシブ対応
- モバイル: ハンバーガーメニュー
- タブレット: 2カラムグリッド
- デスクトップ: 3カラムグリッド

## 🚀 使い方

### ヘッダーナビゲーション

```
┌─────────────────────────────────────────────┐
│ [ロゴ] タスク管理    プロジェクト   [アバター▼] │
└─────────────────────────────────────────────┘
                                    ↓クリック
                        ┌─────────────────┐
                        │ プロジェクト一覧  │
                        │ ログアウト       │
                        └─────────────────┘
```

### 検索・フィルター

```
┌────────────────────────────────────────────┐
│ 🔍 検索...   [ステータス▼]  [ソート▼]       │
└────────────────────────────────────────────┘
  ↓
┌────────────────────────────────────────────┐
│ 5件のプロジェクトが見つかりました  [×クリア]  │
└────────────────────────────────────────────┘
```

### パンくずリスト

```
🏠 プロジェクト一覧 > ECサイトリニューアル
     ↑クリック可能
```
