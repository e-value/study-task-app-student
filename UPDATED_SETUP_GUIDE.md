## 🚀 セットアップ手順

### 1. 依存パッケージのインストール

```bash
cd /Users/M60/Desktop/study-task-app

# Composerパッケージのインストール
composer install

# Node.jsパッケージのインストール
npm install
```

### 2. データベースのセットアップ

```bash
# マイグレーション＆シード実行（一発で完了）
php artisan migrate:fresh --seed
```

### 3. 開発サーバーの起動

**ターミナル1: Laravelサーバー**
```bash
php artisan serve
```

**ターミナル2: Vite（Vue開発サーバー）**
```bash
npm run dev
```

## 🎨 新UIの特徴

### グラスモーフィズムデザイン
- `backdrop-blur-lg` - 背景のぼかし効果
- `bg-white/70` - 透過度70%の白背景
- `border border-white/50` - 半透明のボーダー

### グラデーション
- ヘッダー: `from-blue-600 to-indigo-600`
- ボタン: `from-blue-500 to-indigo-500`
- 背景: `from-slate-50 via-blue-50 to-indigo-50`

### アニメーション
- ホバー時の影の拡大
- スムーズな色の変化
- 作業中ステータスの点滅
